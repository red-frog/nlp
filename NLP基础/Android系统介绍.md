## Andriod系统是什么

- 基于linux的自由开源操作系统
- 分层架构
    - 应用程序层
    - 应用程序框架层(Application Framework)--安卓开发者要掌握的层
        - Activity Manager()
        - Window Manager
        - Content Providers
        - View System
        - Notification Manager
        - Package Manager
        - Telephone Manager
        - Resource Manager(资源管理)
        - Location Manager(位置管理) 
        - XMPP(通信)
    - 系统运行库层
    - linux内核层
    
    
## Android 开发流程

- 开始

- 为每个控件添加实现
    - 业务逻辑的实现
    
- 运行调试

- 打包部署
